# Slick Willy Gee Theme

A dark theme for VS Code with vibrant colors optimized for readability and long coding sessions.

## Installation

1. Open VS Code
2. Press `Ctrl+P` / `Cmd+P`
3. Type `ext install slick-willy-gee-theme`
4. Press Enter
